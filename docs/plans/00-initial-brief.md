# Day 08 · 분수 시각화 슬라이더

## 토픽 (#008)
**콘텐츠 주제·목표**: 분자·분모를 슬라이더로 조작하면 원·막대·수직선 그림이 동시에 바뀌며 두 분수 크기를 한 화면에서 비교.

## 포함 기능
- 분수 2개 슬라이더(분자·분모)
- 원형·막대형·수직선 3종 동시 표현
- 동치분수 자동 표시 (예: 1/2 = 2/4)
- 통분 시뮬레이션 (공통 분모 찾기)
- SVG 캡처 → PNG

## 배제 기능
- 학생 입력 저장·채점
- AI 풀이 설명
- 외부 공유

## 기술 스택
- 단일 `index.html` + vanilla CSS + vanilla JS + SVG
- CDN 의존 0 (헤드리스 환경 친화)
- 빌드 도구 없음

## 저장
- 휘발성 (저장 기능 없음)
