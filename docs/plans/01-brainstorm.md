# Brainstorm · 분수 시각화 슬라이더

원격 무인 세션이므로 AI 단독 판단으로 YAGNI 적용.

## 핵심 사용자 시나리오
1. 교사가 TV 화면에 띄우고 슬라이더로 분자/분모 조작하며 설명 → 학생이 즉시 시각적으로 이해.
2. 학생이 두 분수를 각각 조작하며 크기 비교 직관 습득.
3. "통분" 버튼을 눌러 두 분수의 공통 분모를 자동 찾고 변환 과정 시각화.
4. 동치분수 자동 표시로 1/2 = 2/4 = 3/6 ... 즉시 확인.
5. 현재 화면을 PNG로 저장해 인쇄·판서 자료로 활용.

## 꼭 필요한 기능 (MUST)
1. **분수 2개 독립 슬라이더**: 각각 분자(0~분모), 분모(1~12) 슬라이더. 숫자 입력도 가능.
2. **3종 시각화 동시 표시**: 원형(파이), 막대(가로 바), 수직선(0~1 또는 0~2 범위). 슬라이더 값 변경시 즉시 갱신.
3. **크기 비교 표시**: 두 분수의 대소(`<`, `=`, `>`)를 큼지막하게 표시 + 어느 쪽이 큰지 색으로 강조.
4. **동치분수 자동 나열**: 각 분수 옆에 1/2 = 2/4 = 3/6 형식으로 3~4개 동치분수 표시.
5. **통분 시뮬레이션 버튼**: 클릭시 두 분수를 최소공배수 분모로 통분, 변환 과정을 짧은 애니메이션으로 표현.
6. **SVG → PNG 캡처**: "이미지 저장" 버튼으로 현재 화면 캔버스를 PNG 다운로드.

## 있으면 좋은 기능 (SHOULD)
1. **자연수부분 분리 표시**: 가분수일 때 자동으로 대분수 표기 (예: 7/4 = 1과 3/4).
2. **두 분수 동시 변환 자물쇠**: "잠금" 토글로 두 분수의 분모를 동일하게 묶고 비교만 변형.
3. **초기화·예시 버튼**: 1/2 vs 1/3, 2/3 vs 3/4 등 흔한 비교 케이스 빠른 로드.

## 절대 금지 기능 (MUST NOT)
1. **학생 입력 저장**: localStorage·서버 저장 일체 없음. 휘발성. (개인정보 위험 0)
2. **AI 해설·자동 풀이**: Gemini 호출 없음. 토픽 정의에 명시.
3. **계정·로그인·외부 공유**: 단일 페이지 toy 자체로 충분.
4. **통분 외 사칙연산**: 분수 덧셈·뺄셈·곱셈·나눗셈 풀이 도구화 → scope creep. 본 토픽은 "크기 비교 + 시각화" 한정.
5. **다른 단원 확장 (소수, 백분율)**: 토픽 #008 범위 초과.

## 비기능 요구
- 반응형: 1024px 이상 가로 2열, 그 이하 1열.
- 접근성: 모든 슬라이더 키보드 조작 가능, aria-label, 대비 4.5:1.
- 성능: 모든 변환 60fps, 라이브러리 0 (SVG 직조).
- 정확성: 분수 simplify/통분은 BigInt 불필요(분모 ≤ 12 기준), 정수 GCD/LCM으로 충분.
