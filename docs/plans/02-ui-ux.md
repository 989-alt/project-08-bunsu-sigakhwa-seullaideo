# UI/UX Design · 분수 시각화 슬라이더

## 디자인 시스템: Figma DESIGN.md

Figma 브랜드는 흑백 에디토리얼 캔버스에 큼지막한 파스텔 컬러 블록이 끼어드는 톤. 교실용 학습 도구에 적합:
- 색 블록으로 두 분수를 시각적으로 분리.
- 화이트 캔버스 + 두꺼운 sans 활자 → TV에서도 가독성 우수.
- 학생이 화면을 한 눈에 파싱.

## 색 매핑
| 역할 | 색 (Figma DESIGN.md) | 사용처 |
|---|---|---|
| Canvas | `#ffffff` | 페이지 배경 |
| Ink (primary) | `#000000` | 본문, 헤딩, 라벨 |
| Hairline | `#e6e6e6` | 분할선 |
| Surface-soft | `#f7f7f5` | 패널 배경 |
| Block-lilac `#c5b0f4` | 분수 A 컬러 패널 |
| Block-mint `#c8e6cd` | 분수 B 컬러 패널 |
| Block-lime `#dceeb1` | "동치분수" 라벨 박스 |
| Block-cream `#f4ecd6` | "통분 결과" 영역 |
| Accent-magenta `#ff3d8b` | "더 큰 쪽" 강조 보더, 비교 결과 |
| Semantic-success `#1ea64a` | "통분 완료" 토스트 |

## 타이포
- 시스템 sans 폴백: `-apple-system, BlinkMacSystemFont, "Segoe UI", "Pretendard", "Apple SD Gothic Neo", Roboto, sans-serif` (figmaSans 없음 → 안전 fallback)
- Display(분수 표기): 96px, weight 340, line-height 1.0
- Headline: 26px, weight 540
- Body: 18px, weight 320
- Eyebrow/caption: 12~16px monospace 느낌(`ui-monospace`)

태블릿/모바일 1024px 미만: 분수 A/B 패널이 가로 → 세로 스택, 시각화 3종도 1열 스택.

## 컴포넌트
- **FractionPanel**: 분수 표기(대형 SVG), 슬라이더 2개(분자/분모), 동치분수 표시. 두 개 사이드바이사이드.
- **NumeratorSlider/DenominatorSlider**: 키보드 ←→ 가능, 44x44 터치 타겟, 현재값 표시.
- **VisualPie**: SVG 원, 두 분수를 두 원에 표시 (분수 A는 lilac, 분수 B는 mint).
- **VisualBar**: 가로 막대 2개, 동일 폭(1 단위)을 분모로 나눔.
- **VisualNumberLine**: 0~max(1, 가분수면 2)까지의 수직선에 점 2개.
- **CompareBadge**: 큰 `<` / `=` / `>` 기호 + magenta 강조.
- **ActionBar**: pill 버튼 4개 (통분/예시/초기화/PNG).
- **EquationDisplay**: 통분 후 변환을 텍스트로 명시 (3/4 → 6/8, 1/2 → 4/8).

## 접근성
- 모든 슬라이더: `<input type="range">` 사용 (네이티브 키보드 지원), `aria-label="분수 A 분자"` 등.
- 슬라이더 옆에 항상 현재 수치 표기 (색 외 정보).
- 비교 결과: `<`/`=`/`>` 기호 + 텍스트 "분수 A가 더 큽니다" (스크린리더용 visually-hidden 라벨).
- 모든 인터랙티브: 키보드 포커스시 4px 검정 outline (Figma 톤).
- 색 대비: `#000` on `#ffffff` (21:1), `#000` on `#c5b0f4` (≈ 12:1) — 모두 통과.
- `prefers-reduced-motion: reduce` 존중: 통분 애니메이션 즉시 적용.

## 상호작용
- 슬라이더 변경 → 모든 시각화·동치분수·비교 결과 즉시 갱신 (60fps).
- 통분 버튼: 두 분수를 LCM 분모로 변환, 0.5s ease 모핑. reduce-motion일 땐 즉시.
- PNG 저장: 전체 viz 영역을 SVG → canvas → toBlob → download.
- 예시 버튼: 미리 정의된 4쌍 (1/2 vs 1/3, 2/3 vs 3/4, 3/4 vs 5/8, 4/6 vs 2/3 동치 사례) 중 라운드 로빈.

## 인쇄 (옵션)
- `@media print`: 버튼 숨김, 흰 배경, 색 패널을 옅게.

## 검증 포인트 (테스터에게)
1. 초기 진입: 1/2 vs 1/3 로드, 비교 `>` 표시.
2. 분수 A 분자를 3으로 → 시각화·동치분수 즉시 갱신.
3. 분수 A 분모를 4로 → 시각화 갱신, 가분수 안되게 분자 자동 클램프.
4. 통분 버튼 → cream 박스에 "3/4 = 6/8, 1/3 = ...wait 1/2 example".
5. PNG 저장 → 다운로드 트리거 (테스트 환경에서 download 이벤트 확인).
6. 키보드: Tab으로 모든 컨트롤 이동 가능, ←→로 슬라이더 조작.
